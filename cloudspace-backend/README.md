# CloudSpace Backend

This directory contains the AWS Lambda functions and infrastructure configuration for CloudSpace.

## Project Structure

- `lambdas/`: Individual Lambda function code.
- `api-gateway/`: OpenAPI definitions and route mappings.
- `iam/`: Execution roles and policies.
- `env/`: Environment variable configurations.
- `scripts/`: Deployment and build scripts.

## Deployment

Use the scripts in the `scripts/` directory to build and deploy the Lambda functions.
